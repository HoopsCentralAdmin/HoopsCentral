# Tokenomics

This directory outlines the token economy for HoopsCentral.

The HoopsCentral ecosystem will utilize blockchain-based incentives to reward participation, creativity, and competition.

Key components include:

• Reward distribution for challenge winners  
• Creator tipping using USDC  
• Incentives for community participation  
• Potential governance mechanisms  

The token model is designed to align incentives between players, creators, fans, and platform partners.

Further documentation will outline supply allocation, reward pools, and long-term ecosystem sustainability.
