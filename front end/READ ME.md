# Frontend

This directory contains the frontend interface for the HoopsCentral platform.

The frontend allows users to:

• Upload basketball clips and highlights  
• Participate in skill-based challenges  
• Tip creators using USDC  
• View leaderboards and rankings  
• Discover community competitions  
• Interact with brand-sponsored challenges  

The frontend will support both web and mobile environments and will connect directly to Polygon smart contracts for on-chain reward distribution.

Future iterations will include wallet integrations and seamless Web3 onboarding for new users.
