# HoopsCentral

HoopsCentral is a decentralized basketball platform built on Polygon that rewards players and creators for skill, competition, and community participation.

## Vision
To create a global basketball participation economy where players of all levels can earn value through competition, challenges, and content.

## Core Features
- Skill based basketball challenges
- Creator tipping using USDC
- Tokenized reward system
- Brand sponsored competitions
- Community driven ranking system

## Built On
Polygon blockchain

## MVP
Mobile platform for posting clips, launching challenges, and rewarding performance.

## Future Development
- NFT player highlights
- Global tournaments
- Brand sponsored challenges
